Introduction

Beer styles have a ideal temperature of consumption, this API support a CRUD of beer styles and a search of playlist beer by closer average temperature. A playlist beer has a beer style name and a Spotify playlist that contain the beer style.

Beer API

Collection of CRUD API endpoints used to manipulate beer styles registered in the application. And an option of find playlist beer by average temperature

Get Playlist Beer by Closer Average Temperature

Obtains a specific playlist beer registered in the application by closer average temperature, if has two playlists beer with same average temperature obtain per alphabet order. If don’t find a playlist with beer style name a not found status will be returned.

Sample Request

GET /playlist-beers/beer-styles?temperature=10 HTTP/1.1
Host: localhost:8080
